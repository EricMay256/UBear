# UBear Framework

Personal cross-project Unity library.

## Installation

Add to your project's `Packages/manifest.json`:

```json
{
  "dependencies": {
    "com.ericmay.ubear": "https://github.com/EricMay256/UBear.git#v0.1.4"
  }
}
```

Or via Unity Package Manager → "Add package from git URL..." with the same URL.

## Versioning

Semantic versioning. Pre-1.0, breaking changes can occur on minor bumps.

## Testing

Running tests: Add "testables": ["com.ericmay.ubear"] to your project's Packages/manifest.json to make UBear's tests visible in Unity's Test Runner.
